# Resume Builder

## 📌 Project Description
Resume Builder Pro is a web-based application that helps users create professional resumes easily. It provides a modern UI, multiple templates, and profile photo upload functionality.

---

## 🚀 Features
- Create resume instantly
- Multiple templates (Modern & Classic)
- Profile photo upload
- Live preview
- Download resume as PDF
- Responsive design

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- jsPDF Library

---

## ▶️ How to Run
1. Download or clone the repository
2. Open index.html in any browser
3. Start building your resume
